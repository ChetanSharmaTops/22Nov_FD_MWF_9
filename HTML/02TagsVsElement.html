Tags Vs Element Vs Attribute

Tags

Tags are those which comes in < AngularBrackets>, with the help of tags we can display our content.

Tags are of Two types:-

  1. They are come with Opening Tag and closing tag, for eg. html, head, body, title, h1, p etc
  2. They are self closing Tag or Empty Tag or Void Tag. for eg. meta, img, hr, input etc

Element

Start Tag - Some Content - End Tag

Attribute

They provide some information in starting tag by which behaviour of element will change.
